< hr> vs < br>

Ik kom twee verschillende elementen tegen in deze opdracht, ze lijken op elkaar maar er zitten wel verschillen in:

< hr>

In de opdracht Je eerste HTML pagina, deel 2, punt 5 wordt gevraagd een < hr> element te gebruiken.

< hr> Leeg element waar met CSS verder op gestyled kan worden.

< br>

In de link https://www.w3schools.com/html/html_elements.asp wordt echter de tag < br> besproken.
Als ik google op het element < hr> en < br> zie ik het volgende:

< br> Leeg/ loos element zonder verder extra toepassingsmogelijkheden

Conclusie

CSS maakt het gebruik van < hr> visueel en veelzijdiger.
Het gebruik van een < br> is echt leeg en ziet eruit als een:

spatie.

Bonus: Het element < p> heeft hetzelfde visuele effect als elementen combinatie < br>< br>
